# SCHOOLME101

SCHOOLME101 curriculum content plus an MCP scaffold foundation for educational AI tutor integrations.

## MCP scaffold status

This repository contains **scaffold/configuration artifacts** for MCP servers.

- Included server definitions are templates and integration contracts.
- They are **not proof of live deployed external services**.
- Runtime deployment, credentials, and external service operations are out of scope for this scaffold layer.

## Scaffolded MCP servers

- `education`
- `quiz`
- `homework`
- `automation_admin`
- `retrieval`
- `student_skills_r12` (new Grade R–12 skills orchestration server)
- `github` (new GitHub integration server template)
