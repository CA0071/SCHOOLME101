# Africa Localization Plan (Foundation)

This repository now includes localization-first scaffolding for an Africa-focused tutoring assistant.

## Current foundation

- `policies/multilingual_localization_policy.json`
- Language preferences in `student_subscription.schema.json`
- Android and gateway config templates with low-bandwidth-friendly setup

## Practical next steps

1. Add localized prompt packs per subject and grade.
2. Add glossary files for key CAPS terms in major South African languages.
3. Build quality checks for translation accuracy and educational terminology.
4. Add telemetry for language selection and response quality feedback.

## Guiding principles

- Keep explanations culturally and regionally relevant.
- Support multilingual output while preserving key academic terms.
- Offer concise response mode for low-connectivity users.
